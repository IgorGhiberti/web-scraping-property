# 🏠 Web Scraper Imobiliário (Projeto Educacional)

Este projeto é uma aplicação simples de **web scraping em C# utilizando HtmlAgilityPack**, desenvolvida com o objetivo de praticar:

- Navegação no DOM
- Uso de XPath
- Algoritmo de paginação
- Extração e normalização de dados
- Extension Methods
- Estruturação de dados com modelos tipados

> ⚠️ Projeto desenvolvido exclusivamente para fins educacionais.

---

## 📌 Objetivo

O objeto principal foi fazer um projeto divertido e que treinasse meu raciocínio, abstraindo um pouco a parte de padrões de projeto e focando mais na lógica. Sem uso e auxílio de inteligência artificial.
